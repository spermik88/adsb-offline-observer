# Portable Release Notes

## Для пользователя
- Приложение работает как portable-сборка: запуск идет без установки.
- Live-режим использует встроенный `dump1090`.
- Playback, история и офлайн-карты доступны без интернета.
- Рабочие данные, карты, записи и логи лежат в portable storage, а не внутри versioned-папки приложения.

## Ограничения
- Portable-версия не устанавливает драйверы RTL-SDR.
- Для live-режима нужен заранее подготовленный RTL-SDR и свободный порт `30003`, если не используется внешний SBS-1 источник.

## Где что лежит
- `data/`: база и настройки
- `maps/`: `MBTiles`
- `recordings/`: экспорт и сохраненные данные
- `logs/`: логи приложения и `dump1090`
